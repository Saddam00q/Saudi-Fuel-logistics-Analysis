# 📦 نظام محاكاة وتحليل سلاسل إمداد ومبيعات محطات الوقود (2024 - 2026)

## 📝 وصف المشروع
مشروع مخصص لمحاكاة البيانات التشغيلية واللوجستية لشبكة مكونة من 4 محطات وقود في المملكة العربية السعودية وأسطول نقل وتوزيع، مع مراعاة تقلبات الطلب الموسمية (مثل الأعياد، الإجازات، وأثر الأمطار على طرق السفر السريعة).

## 📊 قواعد العمل المطبقة (Business Rules)
* **الأسعار معتمدة بدقة:** (بنزين 91: 2.18 ريال، بنزين 95: 2.33 ريال، ديزل: 1.79 ريال).
* **سعة شاحنات النقل:** ثابتة وموحدة عند **32,000 لتر** لكل رحلة تشغيلية.
* **تخصيص أسطول النقل:** شاحنتان مخصصتان للبنزين بجميع أنواعه، وشاحنة مخصصة للديزل فقط.
* **المحاكاة الموسمية:** تراجع سحب الديزل أيام الجمع والأعياد، مع قفزات مبيعات البنزين في مواسم العطل والأمطار.
